# LED Куб 3x3x3 на Arduino Uno

Упрощённая версия проекта [8x8x8 LED Cube](https://www.instructables.com/8x8x8-Arduino-LED-Cube/),
адаптированная для начинающих.

---

## Принцип работы

Куб использует **мультиплексирование по слоям** (layer multiplexing):

- **9 столбцов** (аноды LED) подключены к цифровым пинам Arduino через резисторы 220 Ом.
- **3 слоя** (катоды LED) управляются через NPN-транзисторы (2N2222 / BC547).
- Arduino быстро переключает слои (POV — persistence of vision), создавая иллюзию
  одновременного свечения всех 27 светодиодов.

```
     Вид сверху (нумерация столбцов):

         C0   C1   C2
         C3   C4   C5
         C6   C7   C8

     Слои (вид сбоку):
         Layer 2 (верхний)
         Layer 1 (средний)
         Layer 0 (нижний)
```

## Схема подключения

```
                        Arduino Uno
                     ┌──────────────────┐
                     │                  │
    Столбцы          │  D2  ──[220Ω]──── C0 (анод LED)
    (аноды)          │  D3  ──[220Ω]──── C1
                     │  D4  ──[220Ω]──── C2
                     │  D5  ──[220Ω]──── C3
                     │  D6  ──[220Ω]──── C4
                     │  D7  ──[220Ω]──── C5
                     │  D8  ──[220Ω]──── C6
                     │  D9  ──[220Ω]──── C7
                     │  D10 ──[220Ω]──── C8
                     │                  │
    Слои             │  D11 ──[1kΩ]──┐  │
    (катоды           │  D12 ──[1kΩ]──┤  │
     через            │  D13 ──[1kΩ]──┤  │
     транзисторы)    │                │  │
                     │  GND ──────────┘  │
                     └──────────────────┘

    Транзистор 2N2222 (для каждого слоя):

        Arduino Pin ──[1kΩ]──► Base
                               Collector ◄── Катоды слоя (все 9 LED)
                               Emitter ──► GND
```

### Подключение пинов

| Arduino Pin | Назначение           |
|-------------|----------------------|
| D2          | Столбец 0 (анод)     |
| D3          | Столбец 1            |
| D4          | Столбец 2            |
| D5          | Столбец 3            |
| D6          | Столбец 4            |
| D7          | Столбец 5            |
| D8          | Столбец 6            |
| D9          | Столбец 7            |
| D10         | Столбец 8            |
| D11         | Слой 0 (нижний)      |
| D12         | Слой 1 (средний)     |
| D13         | Слой 2 (верхний)     |
| GND         | Эмиттеры транзисторов|

## Сборка куба

### 1. Подготовка светодиодов
- Проверь каждый LED перед пайкой (с резистором 220 Ом от 5V).
- Определи полярность: длинная ножка = анод (+), короткая = катод (−).
- Согни катод (−) каждого LED на 90° — они будут соединяться горизонтально в слой.

### 2. Изготовление шаблона (jig)
- Нарисуй сетку 3×3 на картоне/фанере с расстоянием ~20 мм между центрами.
- Просверли 9 отверстий диаметром 5 мм.

### 3. Пайка слоёв
- Вставь 9 LED в шаблон.
- Спаяй все катоды (−) в горизонтальную сетку с помощью медной проволоки.
- Повтори для 3 слоёв. Проверь каждый слой!

### 4. Соединение столбцов
- Поставь нижний слой на макетную плату.
- Припаяй средний слой сверху, соединяя аноды (+) вертикально.
- Припаяй верхний слой.
- Используй прокладки одинаковой высоты между слоями (~20 мм).

### 5. Подключение к Arduino
- 9 анодов (столбцы) → через резисторы 220 Ом → пины D2–D10.
- 3 катодных слоя → через транзисторы 2N2222 → пины D11–D13.

## Загрузка кода

1. Открой `led_cube_3x3x3.ino` в Arduino IDE.
2. Выбери плату: **Arduino Uno**.
3. Выбери порт.
4. Нажми **Upload**.

## Анимации в коде

| Анимация          | Описание                                          |
|-------------------|---------------------------------------------------|
| `allOn/allOff`    | Включить/выключить весь куб                       |
| `layerWipe`       | Последовательное зажигание слоёв снизу вверх      |
| `columnWipe`      | Последовательное зажигание столбцов                |
| `rain`            | Эффект дождя — случайные капли падают вниз         |
| `spiral`          | Спираль по периметру куба                          |
| `randomBlink`     | Случайное мигание отдельных LED                    |
| `planeBounce`     | Слой «прыгает» вверх-вниз                         |
| `diagonalWipe`    | Диагональная волна                                 |
| `edgesOnly`       | Только рёбра куба                                  |
| `fireworks`       | Простая имитация фейерверка (вдохновлено оригиналом)|

## Сравнение с оригинальным 8x8x8

| Параметр           | 8x8x8 (оригинал)    | 3x3x3 (наш)         |
|--------------------|----------------------|----------------------|
| Кол-во LED         | 512                  | 27                   |
| Пины Arduino       | 64 анода + 8 катодов | 9 анодов + 3 катода  |
| Доп. микросхемы    | Shift registers      | Не нужны!            |
| Транзисторы        | 8                    | 3                    |
| Сложность пайки    | Очень высокая        | Простая               |
| Время сборки       | 20+ часов            | 2–3 часа             |

# LED Куб 3x3x3 на ESP32 (MicroPython)

Упрощённая версия проекта [8x8x8 LED Cube](https://www.instructables.com/8x8x8-Arduino-LED-Cube/),
адаптированная для ESP32 с MicroPython.

---

## Принцип работы

Куб использует **мультиплексирование по слоям** (layer multiplexing):

- **9 столбцов** (аноды LED) подключены к GPIO пинам ESP32 через резисторы 220 Ом.
- **3 слоя** (катоды LED) управляются через NPN-транзисторы (2N2222 / BC547).
- ESP32 быстро переключает слои (POV — persistence of vision), создавая иллюзию
  одновременного свечения всех 27 светодиодов.

```
     Вид сверху (нумерация столбцов):

         C0   C1   C2
         C3   C4   C5
         C6   C7   C8

     Слои (вид сбоку):
         Layer 2 (верхний)
         Layer 1 (средний)
         Layer 0 (нижний)
```

## Схема подключения

```
                        ESP32 DevKit
                     ┌──────────────────┐
                     │                  │
    Столбцы          │  GPIO2  ──[220Ω]── C0 (анод LED)
    (аноды)          │  GPIO4  ──[220Ω]── C1
                     │  GPIO5  ──[220Ω]── C2
                     │  GPIO18 ──[220Ω]── C3
                     │  GPIO19 ──[220Ω]── C4
                     │  GPIO21 ──[220Ω]── C5
                     │  GPIO22 ──[220Ω]── C6
                     │  GPIO23 ──[220Ω]── C7
                     │  GPIO25 ──[220Ω]── C8
                     │                  │
    Слои             │  GPIO26 ──[1kΩ]──┐│
    (катоды           │  GPIO27 ──[1kΩ]──┤│
     через            │  GPIO32 ──[1kΩ]──┤│
     транзисторы)    │                │ │
                     │  GND ──────────┘ │
                     └──────────────────┘

    Транзистор 2N2222 (для каждого слоя):

        ESP32 GPIO ──[1kΩ]──► Base
                              Collector ◄── Катоды слоя (все 9 LED)
                              Emitter ──► GND
```

### Подключение пинов

| ESP32 Pin  | Назначение           |
|------------|----------------------|
| GPIO 2     | Столбец 0 (анод)     |
| GPIO 4     | Столбец 1            |
| GPIO 5     | Столбец 2            |
| GPIO 18    | Столбец 3            |
| GPIO 19    | Столбец 4            |
| GPIO 21    | Столбец 5            |
| GPIO 22    | Столбец 6            |
| GPIO 23    | Столбец 7            |
| GPIO 25    | Столбец 8            |
| GPIO 26    | Слой 0 (нижний)      |
| GPIO 27    | Слой 1 (средний)     |
| GPIO 32    | Слой 2 (верхний)     |
| GND        | Эмиттеры транзисторов|

> **Примечание:** GPIO пины выбраны так, чтобы избежать boot-strapping пинов
> (GPIO 0, 12, 15) и пинов только-ввод (GPIO 34, 35, 36, 39).

## Сборка куба

### 1. Подготовка светодиодов
- Проверь каждый LED перед пайкой (с резистором 220 Ом от 3.3V).
- Определи полярность: длинная ножка = анод (+), короткая = катод (−).
- Согни катод (−) каждого LED на 90° — они будут соединяться горизонтально в слой.

### 2. Изготовление шаблона (jig)
- Нарисуй сетку 3×3 на картоне/фанере с расстоянием ~20 мм между центрами.
- Просверли 9 отверстий диаметром 5 мм.

### 3. Пайка слоёв
- Вставь 9 LED в шаблон.
- Спаяй все катоды (−) в горизонтальную сетку с помощью медной проволоки.
- Повтори для 3 слоёв. Проверь каждый слой!

### 4. Соединение столбцов
- Поставь нижний слой на макетную плату.
- Припаяй средний слой сверху, соединяя аноды (+) вертикально.
- Припаяй верхний слой.
- Используй прокладки одинаковой высоты между слоями (~20 мм).

### 5. Подключение к ESP32
- 9 анодов (столбцы) → через резисторы 220 Ом → GPIO 2,4,5,18,19,21,22,23,25.
- 3 катодных слоя → через транзисторы 2N2222 → GPIO 26,27,32.

## Загрузка кода

1. Прошей MicroPython на ESP32 (если ещё не сделано).
2. Скопируй `main.py` на ESP32 с помощью Thonny, ampy или mpremote.
3. Код запускается автоматически при включении.

## Анимации в коде

| Анимация          | Описание                                          |
|-------------------|---------------------------------------------------|
| `all_on_off`      | Включить/выключить весь куб                       |
| `layer_wipe`      | Последовательное зажигание слоёв снизу вверх      |
| `column_wipe`     | Последовательное зажигание столбцов                |
| `rain`            | Эффект дождя — случайные капли падают вниз         |
| `spiral`          | Спираль по периметру куба                          |
| `random_blink`    | Случайное мигание отдельных LED                    |
| `plane_bounce`    | Слой «прыгает» вверх-вниз                         |
| `diagonal_wipe`   | Диагональная волна                                 |
| `edges_only`      | Только рёбра куба                                  |
| `fireworks`       | Имитация фейерверка (вдохновлено оригиналом)       |
